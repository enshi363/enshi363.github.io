用料  

| **高筋面粉** | 200克 |
| ------------------------------------------------------- | ----- |
| **低筋面粉**                                            | 50克  |
| **糖**    | 20克  |
| **盐**      | 3克   |
| **酵母**   | 3克   |
| **鸡蛋**   | 35克  |
| **水**      | 130克 |
| **黄油**   | 25克  |
| **葱末**    | 适量  |
| **蒜蓉酱用黄油**                                        | 25克  |
| **蒜蓉酱用蒜蓉**                                        | 12克  |
| **蒜蓉酱用盐**                                          | 1克   |



1. 用后油法将面团揉至扩展阶段，室温发酵。发酵直至面团2.5倍大。手指粘高粉戳入中间，抽出手后，空洞基本无变化，不回弹，不变小。
2. 将面团平均分成10份，松弛15分钟
3. 取一块面团，擀成椭圆形。
4. 将两边捏紧。
5. 收口冲下，拍拍扁。
6. 整型好的的面团排入烤盘。进行第二次发酵。
7. 来制作蒜蓉酱：取25克软化的黄油+12克蒜蓉+1克盐，混合均匀。
8. 装入裱花袋，剪一个小口子。
9. 二次发酵好的面团，用小刀在表面划一刀。
10. 在划开的刀口中挤入黄油蒜蓉。
11. 撒上葱末，入烤箱，中层，上下火，180度，20分钟
12. 完成。
